# 🍽️ SaveServe AI

AI-based Food Waste Donation System

## 📌 Project Description

SaveServe AI helps reduce food waste by connecting donors with people in need.
It allows users to donate excess food and track distribution efficiently.

## 🚀 Features

* User login & registration
* Admin panel
* Food donation tracking
* History management
* AI-based decision support

## 🛠️ Tech Stack

* Python (Flask)
* HTML, CSS
* JSON / Text data storage

## 📁 Project Structure

* `app.py` → Main backend
* `templates/` → HTML files
* `data.txt` → Data storage
* `history.json` → Donation history

## ▶️ How to Run

```bash
pip install flask
python app.py
```

Then open:
http://127.0.0.1:5000/
